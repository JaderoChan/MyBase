# My Base

我的"秘密基地"——一个用于存放个人编写或收集的各类文档、规范、模板与脚本的知识库。（持续更新）

## 目录结构

### 📚 references — 参考资料

收集整理的各类速查表与参考文档，包含：

- ASCII 速查表
- LaTeX 符号速查表
- 常见文件头速查表
- 开源协议对比速查表
- 正则表达式语法速查表
- 终端转义序列速查表

### 📏 standards — 规范标准

个人制定的编写与排版规范，包括：

- 编码风格规范
- Markdown 文档空格标准
- 文本尾随句点标准
- 小标题高亮范围标准
- 纯 AI 生成内容标记标准（起草中）

### 📋 templates — 模板

可复用的文件与项目模板：

- **FileTemplate**：通用文件模板（许可证等）
- **ProjectTemplate**：项目脚手架模板
  - `CommonProjectTemplate`：通用项目模板
  - `HeaderOnlyLibraryTemplate`：Header-Only 库模板
  - `LibraryTemplate`：C++ 库模板
  - `QtProjectTemplate`：Qt 应用项目模板

### 🔧 scripts — 脚本

常用工具脚本：

- `create_app_icons`：应用图标批量生成脚本（支持 Windows/macOS）

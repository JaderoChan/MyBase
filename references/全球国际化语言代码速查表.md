## 代码结构（以中文为例）

```plaintext
language-extlang-script-region
  │        │      │      │
  │        │      │      └─ 两位区域码（CN / HK / TW / SG / MO）
  │        │      └──────── 书写体系（Hans 简体 / Hant 繁体）
  │        └─────────────── 扩展语言（很少用）
  └──────────────────────── 语言主码（zh）
```

> 语言主码全小写，区域码全大写

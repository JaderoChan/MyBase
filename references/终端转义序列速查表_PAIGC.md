## 基础约定

| 写法 | 含义 | 备注 |
| --- | --- | --- |
| `ESC` | 转义字符（ASCII 27） | 常写作 `\x1B`、`\033`、`\e` |
| `CSI` | Control Sequence Introducer | 等价于 `ESC [` 或单字节 `0x9B` |
| `OSC` | Operating System Command | 以 `ESC ]` 开头，通常以 `BEL` 或 `ST` 结束 |
| `DCS` | Device Control String | 以 `ESC P` 开头，通常以 `ST` 结束 |
| `ST` | String Terminator | `ESC \\` |
| `BEL` | 响铃字符 | `\a`（ASCII 7） |

## 常见字符串表示

| 场景 | 写法 |
| --- | --- |
| C/C++ 字符串 | `"\x1B[31m"` |
| Bash（ANSI C quoting） | `$'\e[31m'` |
| Bash（printf） | `printf '\033[31mred\033[0m\n'` |
| Python 字符串 | `"\x1b[31m"` |
| PowerShell | `` "`e[31m" ``（`e 为 ESC） |

## C0 控制字符

| 名称 | 十六进制 | 转义 | 含义 |
| --- | --- | --- | --- |
| `BEL` | `0x07` | `\a` | 响铃 |
| `BS` | `0x08` | `\b` | 退格 |
| `HT` | `0x09` | `\t` | 水平制表 |
| `LF` | `0x0A` | `\n` | 换行 |
| `VT` | `0x0B` | `\v` | 垂直制表 |
| `FF` | `0x0C` | `\f` | 换页 |
| `CR` | `0x0D` | `\r` | 回车 |
| `ESC` | `0x1B` | `\e` / `\x1B` | 转义前缀 |

## CSI 基本格式

```text
CSI Ps ; Ps ... <Final>
即：ESC [ 参数1 ; 参数2 ... 终止字母
```

- `Ps` 是数字参数（可省略，省略时使用默认值）。
- 不同终止字母（如 `A`、`m`、`J`）表示不同命令。

## 光标控制（CSI）

| 序列 | 名称 | 作用 |
| --- | --- | --- |
| `ESC [ n A` | `CUU` | 光标上移 `n` 行 |
| `ESC [ n B` | `CUD` | 光标下移 `n` 行 |
| `ESC [ n C` | `CUF` | 光标右移 `n` 列 |
| `ESC [ n D` | `CUB` | 光标左移 `n` 列 |
| `ESC [ n E` | `CNL` | 光标下移 `n` 行并到行首 |
| `ESC [ n F` | `CPL` | 光标上移 `n` 行并到行首 |
| `ESC [ n G` | `CHA` | 光标移到当前行第 `n` 列 |
| `ESC [ r ; c H` | `CUP` | 光标移动到第 `r` 行第 `c` 列（1-based） |
| `ESC [ r ; c f` | `HVP` | 同 `CUP` |
| `ESC [ s` | `SCP` | 保存光标位置（SCO 风格） |
| `ESC [ u` | `RCP` | 恢复光标位置（SCO 风格） |
| `ESC 7` | `DECSC` | 保存光标和属性（DEC 旧式） |
| `ESC 8` | `DECRC` | 恢复光标和属性（DEC 旧式） |

## 擦除与滚动（CSI）

| 序列 | 名称 | 作用 |
| --- | --- | --- |
| `ESC [ 0 J` | `ED` | 清除光标到屏幕末尾 |
| `ESC [ 1 J` | `ED` | 清除屏幕开头到光标 |
| `ESC [ 2 J` | `ED` | 清除整个屏幕 |
| `ESC [ 3 J` | `ED` | 清除整个屏幕和滚动回滚区（部分终端支持） |
| `ESC [ 0 K` | `EL` | 清除光标到行末 |
| `ESC [ 1 K` | `EL` | 清除行首到光标 |
| `ESC [ 2 K` | `EL` | 清除整行 |
| `ESC [ n S` | `SU` | 整页内容上滚 `n` 行 |
| `ESC [ n T` | `SD` | 整页内容下滚 `n` 行 |

## 行/字符编辑（CSI）

| 序列 | 名称 | 作用 |
| --- | --- | --- |
| `ESC [ n L` | `IL` | 插入 `n` 行 |
| `ESC [ n M` | `DL` | 删除 `n` 行 |
| `ESC [ n @` | `ICH` | 插入 `n` 个空白字符 |
| `ESC [ n P` | `DCH` | 删除 `n` 个字符 |
| `ESC [ n X` | `ECH` | 擦除 `n` 个字符（以空格覆盖） |

## 屏幕与缓冲区控制

| 序列 | 含义 | 备注 |
| --- | --- | --- |
| `ESC [ ? 25 l` | 隐藏光标 | 常用于绘制动画 |
| `ESC [ ? 25 h` | 显示光标 | 退出前记得恢复 |
| `ESC [ ? 1049 h` | 切换到备用屏幕 | 类似 `vim`、`less` 的全屏界面 |
| `ESC [ ? 1049 l` | 返回主屏幕 | 恢复原本内容 |
| `ESC [ r` | 重置滚动区域为全屏 | 清除 `DECSTBM` 的限制 |
| `ESC [ top ; bottom r` | 设置滚动区域 | `DECSTBM` |

## OSC（标题、超链接等）

| 序列 | 功能 | 说明 |
| --- | --- | --- |
| `ESC ] 0 ; text BEL` | 设置图标名和窗口标题 | `0` 同时设置图标和标题 |
| `ESC ] 1 ; text BEL` | 设置图标名 | 仅部分终端可见 |
| `ESC ] 2 ; text BEL` | 设置窗口标题 | 最常用 |
| `ESC ] 8 ;; url ST` + `text` + `ESC ] 8 ;; ST` | 终端超链接 | 也可用 `BEL` 替代 `ST` 结束 |

## 示例

### Bash：彩色输出并重置

```bash
printf '\033[1;32mSuccess\033[0m\n'
```

### Bash：清屏并把光标移到左上角

```bash
printf '\033[2J\033[H'
```

### Bash：隐藏光标，结束时恢复

```bash
printf '\033[?25l'
# ... 绘制内容
printf '\033[?25h'
```

### Bash：输出可点击链接（需要支持的终端）

```bash
printf '\033]8;;https://example.com\033\\点击这里\033]8;;\033\\\n'
```

## 兼容性与注意事项

- `xterm`、`iTerm2`、`Windows Terminal`、`GNOME Terminal` 大体兼容 VT100 / ANSI，但细节并不完全一致。
- `\e` 并非所有语言都支持，跨语言时优先使用 `\x1B` 或 `\033`。
- 某些终端不支持真彩色（`38;2;r;g;b`），会回退到近似色。
- 使用 `ESC [ 3 J` 清空回滚区可能在部分终端无效。
- 程序异常退出时可能导致光标未恢复，建议在退出钩子里发送 `ESC [ ? 25 h` 和 `ESC [ 0 m`。
